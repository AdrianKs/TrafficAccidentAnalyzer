What was developed?

The team developed a web application that, in case of an accident, receives data from several sensors installed in cars. Upon transmission success, the application analyzes and classifies the given data. The aim of this is to find dependencies between different pieces of information such as the amount of accidents and the year of construction. The results of this analysis are then displayed in the form of bar charts and a heat map.

Why?

The application provides the possibility to draw immediate conclusions concerning the nature and the severity of accidents. Using the Traffic Accident Analyzer, it is possible to determine what circumstances and places are most susceptible to accidents in order to help prevent these types of situations in the future. Furthermore, paramedics can be given access to collected data to inform themselves about the situation before even arriving at the scene.

How was it developed?

Technological backbone:
  • Java as the app's backbone
    • Development of the app as a web application
    • Development of the components with HTML5, CSS and Javacode
  • Spark Java as web server technology
  • Apache Spark as data transmission service
  • Kafka as data processing and storage technology
  • Python as a tool to create historic data
    • Creation of a script that combines pieces of data to realistic information about accidents

Result

The following slides show the result of the development process:

Analysis of accident data

The first thing the application does upon receiving new data is to classify the accident by analyzing the movement speed and damaged vehicle parts. Possible classifications are a damaged tire, window damage due to colliding objects (e.g. stone chips) and three different types of collisions depending on the amount of damaged parts.

Display of dependencies

When run, the application opens a browser tab that contains the heat map and the previously mentioned bar charts that show the dependencies between different pieces of information and the amount of accidents.
